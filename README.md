# data-analytics-portfolio
Bienvenido a mi portafolio profesional.  
Soy un analista de datos en crecimiento, con conocimientos en **Python, SQL, R, Tableau, estadística y análisis de datos**, enfocado en resolver problemas reales mediante datos y aportar insights accionables para las empresas.

Mi objetivo es desarrollar soluciones analíticas que conviertan datos en decisiones, optimización y crecimiento.

## Habilidades Técnicas

- **Lenguajes:** Python, SQL, R  
- **Librerías:** pandas, NumPy, Matplotlib, Seaborn, scikit-learn  
- **Herramientas BI:** Tableau, Power BI  
- **Bases de datos:** MySQL, PostgreSQL, SQLite  
- **Estadística:** probabilidades, inferencia, A/B tests  
- **Machine Learning:** modelos supervisados, modelos de series temporales  
- **Otros:** Análisis exploratorio, storytelling, limpieza de datos, ETL
